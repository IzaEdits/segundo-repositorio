Comandos Git

Inicialização das Atividades:

  • git fetch --all

    • Sincroniza o repositório local, com o remoto.

  • git pull

    • Faz o download das atualizações do servidor remoto.

Finalização das Atividades:

  • git status

    • Lista as modificações.

  • git add .

    • Adiciona as modificações no servidor local.

  • git commit -m "[texto]"

    • Monta as modificações para enviar ao servidor remoto.

  • git push

    • Faz o upload das modificações do servidor local, para o servidor remoto.
